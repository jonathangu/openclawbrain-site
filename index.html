A local, self-improving knowledge brain for AI agents

ocbrain compiles what your agents did into memory they can trust.

ocbrain ingests the history your OpenClaw, Codex, and Claude agents already produce into a local SQLite ledger, compiles it into source-backed knowledge, decides what worked and what misled, and promotes only the trustworthy parts into an injectable memory — while quietly building a fine-tuning dataset from your own decisions.

As of v0.2 it runs itself: an autopilot on a 15-minute light and hourly heavy cadence replaces the old human approval queue with automatic safeguards — injection scanning, provenance no-clobber, secret scrubbing, a privacy ratchet, tripwire quarantine, and a hash-chained audit trail. Nothing durable becomes injectable unchecked, and nothing leaves the machine.

Don't make each agent remember alone, and don't make a human approve every memory. Give them one evidence-backed brain that labels its own work and guards itself.

Current installed shape

ocbrain runs as a local MCP server for Codex, Claude Code, and OpenClaw, plus an autopilot — 15-minute light passes, hourly heavy passes — that harvests history, compiles knowledge, auto-labels outcomes, promotes trustworthy memory, and mines datasets. All three runtimes read the same brain; retrieval feedback flows back and drives the next promotion cycle.

Runtime tools

brain.search

Search source-backed OpenClaw, Codex, Claude, task, and workspace history without making each runtime keep a separate memory silo.

brain.digest

Expose ledger counts and state so memory quality is inspectable instead of vibes-based.

brain.get

Retrieve reviewed candidates with scope and draft/private controls, instead of injecting raw transcripts blindly.

brain.feedback

Record helpful, used, ignored, irrelevant, or harmful outcomes against a retrieval-use id.

What ocbrain is, and why you can trust it

ocbrain is a local, source-backed knowledge brain for coding agents. Immutable evidence goes in; a compiler turns it into current knowledge; the brain labels that knowledge good or bad from its own outcomes; and memory is a read-only view over only the trustworthy parts. Nothing durable is invented in the dark — every belief traces back to something that actually happened.

Evidence before belief

Raw history is captured as immutable evidence first. Knowledge is never written ahead of the evidence that justifies it.

Labels its own work

A 19-signal taxonomy reads corrections, tests, deploys, and retrieval outcomes to decide which knowledge earned trust — with time decay and confidence.

Promotes only the good

Memory is scored, budget-bounded, and injection-clean. A label flip to bad, or a tripwire, demotes a row immediately.

Safeguards, not gates

Automatic invariants replace the human approval queue: injection scan, provenance no-clobber, secret scrub, privacy ratchet, tripwire quarantine, hash-chained audit.

Safety stance: ocbrain learns autonomously, but it does not act in the world — a self-guarding librarian and compiler, not an agent. Safe & inspectable →

Quick start

git clone https://github.com/jonathangu/ocbrain.git
cd ocbrain
/opt/homebrew/bin/python3.13 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
ocbrain --db data/ocbrain.sqlite mcp
