What changes in practice

OpenClaw runs the agent. OpenClawBrain improves its memory and context selection over time. The difference shows up in three ways.

Fewer repeated mistakes

Corrections stick

When you fix a mistake, OpenClawBrain records what went wrong and why. The next time a similar situation comes up, the agent retrieves the correction instead of repeating the error.

Cleaner prompts

Less noise, more signal

A trained routing function picks only the context that matters for each question — not everything that matches a keyword. Smaller, more precise prompts with better results.

Better recurring workflows

Improves with use

Outcomes and feedback feed back into training. Workflows your agent runs repeatedly get better over time as the routing function learns what context actually helps.

How it works

Three ideas, in plain English.

Knowledge graph

Facts stored as connections, not documents

Instead of a pile of text chunks, knowledge lives in a graph. "This deploy policy depends on that rollback rule" is a real edge, not two documents that share a word. Relationships are first-class.

Learned routing

A trained function picks what to retrieve

When a question arrives, the router walks the graph and selects the most useful nodes. It learns from corrections and outcomes, so it gets better at picking the right context every time.

Structural plasticity

The graph reshapes as things change

Nodes connect, split, merge, and prune under policy control. Stale knowledge fades; active connections strengthen. Ablations show this is the largest single accuracy contributor across every benchmark.

Proof at a glance

Current status

Alpha — v0.1.11

OpenClawBrain is in early alpha. The benchmark results are real; the production surface is not yet hardened. The routing function, graph structure, and plasticity policy are all working and tested. The CLI exposes status and rollback. Bootstrap attach is programmatic (TypeScript). Expect breaking changes before a stable release. Apache-2.0 licensed. Requires Node 20+.

For developers

Install from the registry and connect to your OpenClaw agent. The CLI surface is status and rollback. Bootstrap attach is done programmatically in TypeScript. 

npm install @openclawbrain/openclaw
