Post 4: Operator Guide for Local Embeddings OOTB

March 2026 · v12.2.6+ series

Local embeddings out-of-the-box are the default operating mode for fast, private routing. Teacher models are optional and run asynchronously.

Baseline Flow

openclawbrain init --workspace ./workspace --output ./brain
openclawbrain query "..." --state ./brain/state.json --json
openclawbrain async-route-pg --state ./brain/state.json --apply --json

Operational Notes

  • Keep hot-path routing deterministic and bounded.
  • Run learning jobs on schedule, not inline with user requests.
  • Gate connect/split/merge actions behind shadow evaluation and rollback support.
