Reproduce from the public install lane

Install the plugin, run the CLI against your OpenClaw home, restart the gateway, and verify with one command.

Verified install lane: @openclawbrain/openclaw@0.4.0 plus @openclawbrain/cli@0.4.10 
openclaw plugins install @openclawbrain/openclaw@0.4.0
npx @openclawbrain/cli@0.4.10 install --openclaw-home ~/.openclaw
openclaw gateway restart
npx @openclawbrain/cli@0.4.10 status --openclaw-home ~/.openclaw --detailed

Verification command: npx @openclawbrain/cli@0.4.10 status --openclaw-home ~/.openclaw --detailed

Metadata note: native V2 metadata ships through @openclawbrain/contracts@0.3.5 and is consumed by @openclawbrain/cli@0.4.10. The install path stays plugin + CLI.

Operator note: the plugin install may warn that the manifest id is openclawbrain while the package says openclaw. This warning is cosmetic. The flow above is verified.

Package reference
