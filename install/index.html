OpenClawBrain start here

Install it in four commands.

This is the only path a newcomer needs: install OpenClawBrain, restart the gateway, check status, and save proof if you want it.

Use this same path later for upgrades and repairs too.

See the commands Need help?

The commands

npx @openclawbrain/cli@0.4.43 openclawbrain install --openclaw-home ~/.openclaw
openclaw gateway restart
npx @openclawbrain/cli@0.4.43 openclawbrain status --openclaw-home ~/.openclaw --detailed
npx @openclawbrain/cli@0.4.43 openclawbrain proof --openclaw-home ~/.openclaw
Current release: OpenClawBrain 0.4.43.

What good looks like

Status

STATUS ok

The runtime is healthy for the home you selected.

Load proof

loadProof=status_probe_ready

OpenClawBrain is loaded, not just sitting on disk.

Clean install

surface ... converge=converged

The install is in sync and ready to trust.
