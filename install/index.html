Install

Install in a minute.

OpenClawBrain 0.2.21 ships the route-policy-v3 learning loop: learned routing, calibrated abstention, shadow replay, and local proof surfaces. Use the same command for a fresh install or upgrade. This is the short path; GitHub has the full operator guide.

Install

openclaw plugins install clawhub:openclawbrain@0.2.21 --force
openclaw plugins enable openclawbrain
openclaw gateway restart

Requires OpenClaw 2026.5.2 or later.

Tiny proof loop

openclaw plugins inspect openclawbrain --runtime
openclaw doctor
# /plugins/openclawbrain/proof?limit=10
# /plugins/openclawbrain/search?query=pnpm&limit=10
# /plugins/openclawbrain/route-policy
